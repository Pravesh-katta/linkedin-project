Session-based LinkedIn collection

Search LinkedIn posts across all U.S. states and review them in one place.

Expands one keyword like python developer into state-based searches like python developer "TX" and python developer Texas — filtered by Posts · Latest · Past 24 hours. Upload your resume to score posts against your skills.

Before you run searches

  1. Install dependencies and Chromium.
  2. Run python scripts/bootstrap_linkedin_session.py.
  3. Log in to LinkedIn manually once.
  4. Return here and save the session.
